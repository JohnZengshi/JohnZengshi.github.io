>## localtion对象常用的属性、方法

+ 页面跳转

```bash
location.href = "页面地址"
```
+ 页面刷新

```bash
location.reload();
```
+ 获取get请求的参数

```bash
loaction.search
```
+ 获取锚点/锚名

```bash
location.hash
```

+ 获取协议的名称

```bash
location.protocol
```

+ 获取路径名称

```bash
location.pathname
```
